# Huyack.js

Introducing Huyack.js - first-class
Huyack Driven Development
framework for Zayebis applications.

# Usage
##### Initialization
```javascript
let huyack = new Huyack();

huyack.huyack('и в продакшен');
```
##### Prettify
```javascript
let huyack = new Huyack();

huyack.make.zayebis();
huyack.make.moderate.zayebis();
huyack.make.maximum.zayebis();
```
